# CAJA — Interactive Lookbook

An interactive brand-direction lookbook for **CAJA (Carolina's American Judo)**, Matthews, NC.
Its purpose: help the team agree on **one aesthetic direction** to prototype for the website redesign.

## View it

Live site (GitHub Pages): **https://dxlachx.github.io/caja-lookbook/**

Or open `index.html` locally in any browser.

## The three directions

| Direction | Look | Voices |
|---|---|---|
| **Old North** | Collegiate-athletic — cream, navy & athletic red, varsity lettering | Stadium · Varsity Block · Federation |
| **Newsprint** | Editorial black & white, one red accent | Swiss · Slab · Tabloid |
| **High-Key Sport** | Bright modern athletic — near-white, deep ink & electric blue | Studio · Performance · Power |

Each direction shows a shared kit (logo, type, buttons, mobile), a **homepage-layout placeholder**, and three "voice" treatments. Use the ★ on any voice to mark favorites.
